# Power BI HR Analytics Dashboard

Created an interactive HR Analytics Dashboard using Power BI to analyze employee attrition and workforce insights.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Dashboard Includes
- Employee attrition analysis
- Department-wise insights
- Age and gender distribution
- Job role analysis
- Interactive filters and visuals
